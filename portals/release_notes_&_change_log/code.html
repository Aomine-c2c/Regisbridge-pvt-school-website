v4.2.0

New Feature

Released on Oct 24, 2023 • Deployed by System Admin

Dormitory Mobile App

New Dormitory QR Check-in Module

Boarding staff can now use the mobile app to scan student ID cards for rapid check-in/check-out during curfew hours. This update includes real-time syncing with the central attendance dashboard.

  • Added QR scanner to iOS and Android staff apps.
  • New "Curfew Report" available in the Dormitory dashboard.
  • Offline mode support for dorms with poor Wi-Fi connectivity.

Gradebook Calculation Precision Patch

API Authentication Protocol Update

We are deprecating legacy token authentication (v1) in favor of OAuth 2.0 flows. This is a mandatory update for all schools using custom integrations with the EduManage API.

warning Action Required: IT Admin
